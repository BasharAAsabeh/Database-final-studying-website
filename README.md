# Database Finals Study Desk

Static single-page study website powered by `studyData.js`.

Pages:

- Roadmap
- Lab Manuals
- Theoretical Chapters
- Exam Quiz

## Run

From this folder:

```bash
python3 -m http.server 8000
```

Open:

```text
http://localhost:8000
```

The study pages use local structured data in `studyData.js`, and the quiz uses the curated question bank in `app.js`.
